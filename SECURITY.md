# Security Policy

## Reporting a Vulnerability

To report a security vulnerability, please email **<security@dazpilot.app>**.

Do not file a public GitHub issue for security-related issues.

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Scope

- The Tauri application and its Rust backend
- The Daz Studio bridge C++ plugin
- Authentication and API key handling for AI providers
- Script execution and approval mechanisms
